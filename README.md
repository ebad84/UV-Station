\# UV Station



A compact UV curing station designed for PCB and electronics work.



!\[UV Station 3D Preview](https://raw.githubusercontent.com/ebad84/UV-Station/master/3D\_PCB1\_1.png)



\## Overview



UV Station is a small PCB-based UV light source designed for curing UV-sensitive materials such as:



\- UV solder mask

\- UV resin

\- PCB coating materials

\- Other UV-curable compounds



The design focuses on simplicity, low power consumption, and easy accessibility using a standard USB power source.



\## Features



\- 20 × UV 5mm LEDs

\- 20 × 100Ω current limiting resistors

\- USB powered design

\- XH 4-pin connector for power input

\- Compatible with:

&nbsp; - USB chargers

&nbsp; - Mobile phone adapters

&nbsp; - USB ports

&nbsp; - Power banks



\## Electrical Specifications



| Parameter | Value |

|---|---|

| Input Voltage | 5V USB |

| LED Count | 20 UV LEDs |

| Current Limiting | Individual 100Ω resistor per LED |

| Maximum Current | ~400mA |

| Power Consumption | ~2W |



\## Applications



This board can be used for:



\- PCB UV solder mask curing

\- UV resin curing

\- Electronics prototyping

\- DIY PCB manufacturing

\- Educational projects
